# h-reader

Medical image reader utilities for DICOM / NIfTI using SimpleITK.

## Features
- DICOM / NIfTI loading
- Orientation standardization (LPS / RAS)
- Resampling and transforms
- NIfTI export

## Installation
### From source
```bash
git clone https://github.com/yhchoi-data/h-sitk-reader.git
cd h-reader
pip install .
```

### From pypi
```bash
pip install h-sitk-reader
```
